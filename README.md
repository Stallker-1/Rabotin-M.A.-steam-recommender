# 🎮 Steam Game Recommender System

Рекомендательная система игр на основе реальных данных Steam с веб-интерфейсом.

## 📋 О проекте

Система позволяет пользователям:
- 🔍 **Искать игры по текстовому описанию** (например, "стратегия с зомби")
- 🏷️ **Получать рекомендации по жанрам**
- 👤 **Регистрироваться и сохранять историю запросов**

## 🏗️ Архитектура

- **Flask** — веб-интерфейс, аутентификация, UI
- **FastAPI** — ML сервис с рекомендательной моделью
- **SQLite** — хранение пользователей и истории

## 🛠️ Технологии

- Python 3.11
- Flask + FastAPI
- SQLAlchemy + SQLite
- Docker + Docker Compose
- Pandas, NumPy
- TF-IDF для поиска

## 🚀 Быстрый старт

### Требования
- Docker и Docker Compose

## 📖 Использование
- Регистрация — создайте новый аккаунт
- Вход — войдите в систему
- Поиск — введите описание игры (например, "стратегия с зомби")
- Рекомендации — получите список игр с картинками и процентом совместимости
- История — просмотрите все свои запросы

### Запуск

```bash
# Клонирование репозитория
git clone https://github.com/Stallker-1/Rabotin-M.A.-steam-recommender.git
cd Rabotin-M.A.-steam-recommender

# Запуск всех сервисов
docker-compose up --build

# Доступ к сервисам
